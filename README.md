# ♠️ React Blackjack Game ♦️

เกมไพ่แบล็คแจ็ค (Blackjack) ที่สร้างด้วย **React** และ **Tailwind CSS v4** พร้อมระบบการเล่นพื้นฐานและดีไซน์ที่สวยงามรองรับการเล่นบนทุกอุปกรณ์

---

## 🚀 ฟีเจอร์หลัก (Features)
* **ระบบสุ่มไพ่:** สุ่มไพ่สำรับมาตรฐาน 52 ใบ
* **Logic การคำนวณคะแนน:** คำนวณแต้มไพ่อัตโนมัติ (J, Q, K = 10 แต้ม)
* **ระบบเจ้ามือ (Dealer AI):** เจ้ามือจะจั่วไพ่เพิ่มเรื่อยๆ จนกว่าแต้มจะมากกว่าหรือเท่ากับ 17 ตามกติกามาตรฐาน
* **Responsive Design:** แสดงผลสวยงามทั้งบนคอมพิวเตอร์และมือถือด้วย Tailwind CSS
* **Dynamic Assets:** ดึงรูปภาพไพ่แสดงผลตามค่าที่สุ่มได้

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
* [React.js](https://react.dev/) - Frontend Framework
* [Tailwind CSS v4](https://tailwindcss.com/) - Styling
* [Vite](https://vitejs.dev/) - Build Tool
